GIT: Almacenamiento virtual de un proyecto

crear repositorio nube:

 - https://github.com/freddyfernandez/RamasAngular.git

crear repositorio local y subirlo a la nube


 - git init

 - git add .

 - git commit -m "first commit"

 - git remote add origin https://github.com/freddyfernandez/RamasAngular.git(la primera vez)

 - git push -u origin master(la primera vez)


crear ramas de trabajo:

 rama area1: 

- git checkout -b develop : crea y cambia de master a develop

- git push origin develop

- git checkout master: regresa de develop a master

 rama area2:

- git checkout -b devlocal

- git push origin devlocal


Subir nuevos cambios al repositorio en la nube

- git add .

- git status

- git commit -m "comentario"

- git push origin branch-fernandez"(mi rama)

- git pull

- git merge devlocal --no-ff(otra rama)


Migrar Rama:

- merge: siver parsa unir dos ramas

- >>>>HEAD : la rama actual 

- no hacer merge sin bajar los cambios de la nube: hacer pull

- git merge "rama padre" --no-ff


Copiar rama

-git pull origin devlocal

-git merge devlocal --no-ff

-git checkout devlocal(entrar a la rama)

-accept current changes


Ver historial de cambios

- git log

- regresar a un commit anterior: git reset --hard HEAD~1
